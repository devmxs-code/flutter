# Testes Main

Coleção de projetos de teste Flutter e amostras de código.

## Projetos

- **cardapio.simple** - Aplicação de menu simples
- **cardapio2.simple** - Implementação alternativa de menu
- **test.com.efec.gradient** - Projeto de teste de gradiente
- **test.drawer.appbaa.multicolors.zuiera** - Teste de drawer e app bar multicolorido

## Como Começar

Cada projeto é um arquivo Dart independente. Para executar:

1. Navegue até a pasta do projeto específico
2. Execute o arquivo Dart:
   ```bash
   flutter run <main.dart>
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Dart
