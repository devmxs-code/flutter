## Como Começar

Cada projeto é uma aplicação Flutter independente. Para executar qualquer projeto:

1. Navegue até o diretório do projeto:
   ```bash
   cd <nome-do-projeto>
   ```

2. Instale as dependências:
   ```bash
   flutter pub get
   ```

3. Execute a aplicação:
   ```bash
   flutter run
   ```

## Requisitos

- Flutter SDK (>=2.17.0)
- Dart SDK
- Android Studio / VS Code com extensões Flutter
- Simulador iOS / Emulador Android ou dispositivo físico
