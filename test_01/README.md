# Teste 01

Projeto de teste Flutter inicial para aprendizado e experimentação.

## Funcionalidades

- Estrutura básica de aplicação Flutter
- Configuração de projeto de aprendizado

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
