# App de Teste

Aplicação Flutter de teste geral para desenvolvimento e experimentação.

## Funcionalidades

- Estrutura básica de aplicação Flutter
- Configuração de ambiente de testes

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
